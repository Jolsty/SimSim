Per flashare ESP8266 (esp-01)

	- premi RST mentre tieni premuto GPIO0 a GND
	- Connetti RX - blu (esp) con RX(arduino pin 0); TX - giallo (esp) con TX(arduino pin1)
	- Arduino IDE: use board settings like in the image in current directory.
	- Carica
	
Per normal mode (esp-01)

	- Connetti RX - blu (esp) a pin A3; TX - giallo (esp) a pin A2(giallo)
	- premi RST (senza premere GPIO0 a GND)
